## Run your own instance

The Desk is built with Laravel, Inertia + Vue 3, Laravel Reverb (WebSockets),
and Meilisearch. The production stack ships as Docker containers you run
yourself — no SaaS, no per-seat pricing, your data on your infrastructure.
